Creation of git Repository in Desktop app.

GitHub is an online; browser based distributed version control system for software developers using the Git revision control system. The service provides free public repositories, issue tracking, graphs, code review, downloads, wikis, collaborator management, and more. There are also social networking elements of the service including feeds, the ability to follow other users, and network related graphs. This article will walk you through the process of creating a repository in GitHub

We can actually create a repository in two ways. That is either by using GitHub desktop app or directly we can create in online github account. So in this page we are going to explain how to create a repository in Github.

The following is one way to create a Github Repository in Github Desktop app.

  • Go to the GitHub Desktop.
  • Click on Add in current Repository and then click on create repository.
  • Then give the name of the repository and checkin the box to have a README file.
  • Then click on Create Repository.
  • Now the created repository is currently available in local machine only. So inorder to share and collaborate with others we have to publish it.
  • Uncheck the private mode box if you want it to be public and Click on Publish repository.

Creation of git Repository in Github online.

The following is the other way to create a repository in github account.

  1. Click on New near Repositories and then give the name of the repository.
  2. Then select whether you want it as private or public and checkin that box.
  3. Now click on Create Repository Create repository .
  4. Now clone the created repository by clicking on Clone or Download
